Perguntas Frequentes

O formato JPEG usa um algoritmo de compressão com perda: ao reduzir a qualidade, detalhes de alta frequência (bordas finas, texturas complexas) são descartados, diminuindo o tamanho do arquivo. Esta ferramenta usa o elemento Canvas do navegador para recodificar a imagem com o nível de qualidade escolhido — tudo localmente, sem upload.
Para uso na web, valores entre 70 e 85 oferecem ótimo equilíbrio entre tamanho e qualidade visual. Para impressão ou edição posterior, prefira 90 ou mais. Para thumbnails e miniaturas onde o tamanho importa mais, valores entre 40 e 60 costumam ser suficientes.

Esta ferramenta aceita arquivos JPEG/JPG como entrada e gera a saída também em JPEG. Para converter PNG ou WebP para JPEG com compressão, você pode usar nossa ferramenta de conversão de imagens. Note que converter PNG para JPEG pode eliminar a transparência.
