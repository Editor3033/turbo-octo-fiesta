Política de Privacidade

Última atualização: 7 de abril de 2026

Visão Geral

O ToolFlow está comprometido em proteger sua privacidade. Esta política explica como lidamos com dados quando você usa nosso site e ferramentas.

Resumo: Todas as nossas ferramentas processam dados localmente no seu navegador. Nós não coletamos, armazenamos ou transmitimos nenhum conteúdo que você insere nas nossas ferramentas.

Processamento de Dados

Todas as ferramentas do ToolFlow operam inteiramente dentro do seu navegador usando JavaScript do lado do cliente. Isso significa:

  • Seus textos, imagens e arquivos nunca são enviados para nenhum servidor
  • Todo o processamento acontece localmente no seu dispositivo
  • Seus dados nunca saem do seu navegador
  • Nós não podemos ver ou acessar nenhum conteúdo que você insere

Cookies e Armazenamento Local

Podemos usar o armazenamento local do navegador para salvar suas preferências (como configurações de tema ou uso recente de ferramentas) para sua conveniência. Esses dados permanecem no seu dispositivo e nunca são transmitidos para nossos servidores.

Análises

Podemos usar análises amigáveis à privacidade para entender padrões gerais de uso (visualizações de página, ferramentas populares). Esses dados são anônimos e agregados — nunca rastreamos usuários individuais ou coletamos informações pessoais.

Serviços de Terceiros

Algumas ferramentas podem carregar bibliotecas externas (ex.: geração de QR code) de CDNs. Estas são bibliotecas de código aberto padrão amplamente utilizadas. Não usamos redes de publicidade de terceiros nem vendemos dados para terceiros.

Links Externos

Nosso site pode conter links para sites externos. Não somos responsáveis pelas práticas de privacidade desses sites. Recomendamos revisar suas políticas de privacidade antes de compartilhar informações pessoais.

Privacidade de Crianças

O ToolFlow não coleta intencionalmente nenhuma informação pessoal de crianças menores de 13 anos. Nossas ferramentas são projetadas para público geral e não exigem nenhuma informação pessoal para uso.

Alterações nesta Política

Podemos atualizar esta política de privacidade periodicamente. Quaisquer alterações serão publicadas nesta página com uma data atualizada. Encorajamos você a revisar esta página periodicamente.
