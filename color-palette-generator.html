Gerador de Paleta de Cores

Gere paletas de cores bonitas e harmoniosas. Pressione a barra de espaço ou clique em gerar para novas paletas. Clique em qualquer cor para copiar.

🎨 Pressione ESPAÇO para uma nova paleta

FAQ

Aleatório: Cores completamente aleatórias. Análogo: Cores adjacentes no círculo cromático. Complementar: Cores opostas entre si. Triádico: Três cores igualmente espaçadas. Monocromático: Diferentes tons de uma mesma matíz.
Clique em qualquer amostra de cor para copiar seu valor HEX para a área de transferência. Você também pode usar o botão Copiar HEX para copiar todas as cores de uma vez.
